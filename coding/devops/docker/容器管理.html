$ docker exec -it ngx0 bash/sh/csh

# 查看容器文件挂载


容器的创建和重启会到宿主机中把这三个文件挂载出来，所以修改容器中的这三个文件，是不生效的。 重启和创建容器，修改的内容就丢失了。
